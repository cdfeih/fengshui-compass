# 风水罗盘 · 家居指南 🧭

普通人看得懂、用得上的风水罗盘工具。

## 功能

- 🧭 **实时罗盘**：调用手机陀螺仪，精准测量朝向（二十四山方位）
- 📊 **户型分析**：输入房屋信息，自动生成风水评分
- 🔧 **问题化解**：20个常见风水问题的识别和解决方案
- 📚 **知识百科**：10篇通俗易懂的风水基础科普

## 如何使用

### 手机用户
1. 浏览器打开本页面
2. 点击浏览器菜单 → "添加到主屏幕"
3. 即可像App一样使用（支持离线）

### 桌面用户
桌面端使用模拟罗盘，拖动滑块即可旋转方向查看解读。

## 技术栈

React 19 + TypeScript + Vite + PWA
